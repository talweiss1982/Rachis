Introduction
###############################################################################
The tailfeather demo server purpose is to illustrate the use of RAFT consensus 
algorythem using Rachis and Voron as the implementation for the raft protocol 
and storage.

Requirments 
###############################################################################
Voron storage - https://github.com/ayende/ravendb/tree/v3.5/Raven.Voron
Rachis - https://github.com/ayende/ravendb/tree/v3.5/Rachis
typescript for VS2015- https://www.microsoft.com/en-us/download/details.aspx?id=48593

ussage
###############################################################################
Run TailFeatherDemoServer.exe, it should start listening to port 9091.
Open the browser at http://localhost:9091 and the UI should present itself.
